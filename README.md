# unsplash clone project just for education

## start app in development mode
```
yarn run start
```

## test app 

```
yarn run test
```

## run test in watch mode

```
yarn run test-w
```

## build project in production mode

```
yarn run build
```

## build and run server

```
yarn run build-and-run-server
```

## run server only

```
yarn run run-server
```
